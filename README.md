# 📝 MERN To-Do List App

A simple full-stack To-Do List application built using the MERN stack (MongoDB, Express, React, Node.js).

---

## 🚀 Features

- Add new tasks
- View all tasks
- Data stored in MongoDB
- Simple and clean UI

---

## 🛠️ Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB (Compass / Local)
